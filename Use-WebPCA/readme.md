## 注意これを単に実装しても動作しません

このプログラムはリポジトリに含まれるswept-sine法のアプリを実装したのち、ご自身のてでfirebaseと連携するプログラムを作る必要があります。

具体的には計測したインパルス応答のデータをそのままfirebase RTDBに送信してください。（送信データ構造の例をsamleディレクトリに置いておきます）
